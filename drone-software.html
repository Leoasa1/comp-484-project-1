Betaflight Software

Betaflight is flight controller software (firmware) used to fly multi-rotor craft and fixed wing craft. This fork differs from Baseflight and Cleanflight in that it focuses on flight performance, leading-edge feature additions, and wide target support.

Install Betaflight Configurator

In order to setup and configure Betaflight, you need the Betaflight Configurator. You can download it from here.

Older (F4 based) Flight Controllers

For older flight controllers (BRAINRE1, RADIX, RADIX LI), use the Betaflight configurator to flash the firmware as follows.

  1. Download the hex file for your flight controller from the Logo Customizer page. Simply click on your favorite logo (or make your own!) and download the hex file for your flight controller (“RADIX” for RADIX, “BRAINRE1” for RE1, etc.). Alternatively, you can find older and testing releases on the GitHub release page.
  2. Start the Betaflight Configurator and click on “Firmware Flasher” on the left side.
  3. On the bottom, click on “Load Firmware [Local]” and select the hex file you downloaded in Step 1.
  4. On the bottom, click on “Flash Firmware”. This should initiate the flashing process. If it doesn’t work (e.g. if you get a "Could not connect to bootloader" error), you may have a driver problem. See below.

Fixing Driver Problems

If flashing doesn’t work and you get a “not connected to bootloader” error, you most likely have a driver problem. On Windows use the driver fixer” to resolve this problem. See this guide for more information.
